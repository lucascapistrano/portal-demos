    Twitter 'Java' Tweets. The server side polls Twitter every 20 seconds with the search query 'java'

Environments

  • Apache AJP Tomcat
  • Apache HTTP BIO Tomcat
  • Apache HTTP NIO Tomcat
  • HAProxy HTTP BIO Tomcat
  • HAProxy HTTP NIO Tomcat

Websocket currently only works with HAProxy and BIO Connector
