Environments

  • Apache AJP Tomcat
  • Apache HTTP BIO Tomcat
  • Apache HTTP NIO Tomcat
  • HAProxy HTTP BIO Tomcat
  • HAProxy HTTP NIO Tomcat

Websocket currently only works with HAProxy and the BIO Connector. This web application is deployed on a Tomcat that restarts every hour (xx:42)
